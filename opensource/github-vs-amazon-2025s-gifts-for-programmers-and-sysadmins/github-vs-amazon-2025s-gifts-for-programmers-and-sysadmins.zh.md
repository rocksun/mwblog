## 程序员的圣诞节

Google 搜索“送给程序员的礼物”现在会指向亚马逊的一个特殊页面，提供诸如[Linux 填字游戏](https://amzn.to/4oVxSwq)之类的建议。

但是哪个程序员能抵抗一本名为[《程序员烹饪指南》](https://amzn.to/4oWpGw8)的书呢？

“每道食谱都以不同编程语言的源代码形式编写，”书的描述解释道，“这不仅美味，而且具有教育意义。”从披萨吐司到番茄椰子汤——从 Rust 到 Ruby——作者 Richard Wurzer 敦促你“系上围裙，打开命令行。”

![《程序员烹饪指南》亚马逊样品截图](https://cdn.thenewstack.io/media/2025/12/aa8357eb-screenshot-from-amazon-sample-of-cooking-for-programmers.png)

用代码熟悉的舒适感来转换令人生畏的厨房琐事——就像这份烤蔬菜食谱中的JSON：

`name: "peppers",  
quantity: 100,  
unit: "g"`

这本书充满了色彩缤纷的食谱——这得益于语法高亮，将字母变成了红色、蓝色和绿色。书中还有真实的烹饪说明，但以字符串数组的形式呈现，食材和说明最终都作为消息记录到控制台。

有Java、JavaScript、Python、Perl、PHP、TypeScript等等。作者 Wurzer 建议：“让我们开始烹饪，享受乐趣。”

甚至还有一份意式浓缩咖啡的食谱。

不过，如果这些还不够满足你生活中的程序员，那就给他们买一支[带有他们最喜欢气味的蜡烛](https://amzn.to/4iRMvzw)吧。
