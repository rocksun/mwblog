Cloudflare收购了开源JavaScript Web框架Astro的团队。Astro以其高性能和内容驱动的网站构建能力而闻名，未来将继续保持开源。Cloudflare此举旨在加速Astro的发展，并支持开源生态。

> 译自：[Cloudflare Acquires Team Behind Open Source Framework Astro](https://thenewstack.io/cloudflare-acquires-team-behind-open-source-framework-astro/)
> 
> 作者：Loraine Lawson

[Cloudflare](https://thenewstack.io/cloudflare-for-ai-helps-businesses-safely-use-ai/) 周五宣布将收购Astro Technology Company团队，该团队负责开源 [JavaScript Web 框架 Astro](https://github.com/withastro/astro)。

Astro 被IKEA、Unilever、Visa 和 OpenAI 等主要品牌用于构建快速、内容驱动的网站。

Astro 团队将作为 Cloudflare 员工加入。Cloudflare 在一份 [关于收购 Astro 的新闻稿](https://www.cloudflare.com/press/press-releases/2026/cloudflare-acquires-astro-to-accelerate-the-future-of-high-performance-web-development/) 中宣布，为确保其长期增长和发展，[Astro 本身将保持开源](https://github.com/withastro/astro)。

Cloudflare 的声明指出，搜索引擎优先考虑快速加载和简洁的页面。严重依赖 JavaScript 进行初始渲染的网站通常难以提供所需的速度，这会阻碍搜索排名和客户转化。

Astro 页面只提供在浏览器中显示页面所需的代码。这部分归因于其在2021年引入的 [岛屿架构](https://thenewstack.io/astros-journey-from-static-site-generator-to-next-js-rival/)。Astro 的岛屿允许开发者创建交互式客户端组件的“岛屿”，而页面的大部分内容则以 HTML 静态生成。[服务器岛屿](https://thenewstack.io/how-astro-and-its-server-islands-compare-to-react-frameworks/) 将相同的架构扩展到服务器端。

Astro 还是 UI 无关的，这意味着尽管它有自己独立的引擎，但它允许开发者引入来自 React、Svelte、Vue 和其他框架的组件。

根据 Cloudflare 的说法，这使得 Astro 成为构建针对速度优化的、高性能、内容驱动型网站的首选。

## 投资开源工具

Astro 的前 CTO [Fred Schott](https://www.linkedin.com/in/fredkschott/) 与 Cloudflare 高级产品经理 [Brendan Irvine-Broque](https://www.linkedin.com/in/brendanirvinebroque/) 共同撰写了一篇 [关于此消息的 Cloudflare 博客文章](https://blog.cloudflare.com/astro-joins-cloudflare/)。在文章中，他们指出，基于 Cloudflare 构建的平台，例如 Webflow Cloud 和 Wax Vibe，使用 [Astro 为客户在其平台上构建和部署的网站提供支持](https://thenewstack.io/how-to-build-and-deploy-a-basic-site-using-astro-and-netlify/)。

Schott 现在是 Cloudflare 新兴技术与孵化 (ETI) 部门的高级工程经理。

Schott 和 Irvine-Broque 周五写道：“在过去的几年里，我们看到各种各样的开发者和公司使用 Astro 来构建网络应用。在 Cloudflare，我们也使用 Astro——用于我们的开发者文档、网站、登录页面等等。”

Schott 和 Irvine-Broque 补充说，此次收购将使团队能够“加倍努力”，将 Astro 打造成内容驱动型网站的最佳框架。

Cloudflare 联合创始人兼首席执行官 [Matthew Prince](https://www.linkedin.com/in/mprince/) 在公司关于此次收购的声明中表示：“保护和投资 [开源工具](https://thenewstack.io/what-to-do-when-critical-open-source-projects-go-end-of-life/) 对于一个正常运行、自由开放的互联网的健康至关重要。” “通过收购这支才华横溢的团队，并致力于发展这个在速度和性能方面最具影响力的框架之一，我们将确保 Astro 不仅在今天，而且在未来多年都将是内容驱动型网站的最佳 Web 框架。”

就 Astro 而言，Schott 写道，Astro 团队将能够加速该框架的开发，并实现更大规模的推广。
