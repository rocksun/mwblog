维修协会](https://www.repair.org/history)是一个美国倡导团体，致力于争取消费者的“维修权”。该组织认为，如果你购买了某样东西，你就应该能够修理它。然而，消费和工业制造公司[越来越不情愿](https://www.forbes.com/sites/joshmax/2025/01/02/your-new-car-isnt-yoursautomakers-increasingly-charge-subscription-fees/)支持这一理念。

[维修组织](https://thenewstack.io/roasting-the-consumer-electronics-shows-worst-products/)（Repair.org）[连续](https://thenewstack.io/ces-2023-robots-astronauts-schwarzenegger-and-flying-cars/)第[五](https://thenewstack.io/troubling-tech-trends-the-dark-side-of-ces-2024/)年[举办](https://thenewstack.io/ces-2025-worst-in-show-betas-for-a-dystopian-future/)其备受瞩目的年度活动，旨在评选“CES最差产品”，这是一项旨在吸引媒体对每年一月举行的全球最大消费电子展关注的宣传活动。

该[活动](https://www.worstinshowces.com/)得到了《消费者报告》、PIRG、电子前沿基金会（EFF）和iFixit等知名消费者团体的支持，旨在展示CES上“最不注重隐私、最不安全、最难维修和最不环保的小工具”。

维修协会执行董事[Gay Gordon-Byrne](https://www.linkedin.com/in/gay-gordon-byrne-b167855)在[2026年国际消费电子展](https://www.ces.tech/)上表示：“在我看来，CES比以往任何时候都更像是一场营销炒作。许多尚未存在的产品正在被宣布，因此消费者最基本的问题都无法得到提问或回答。”

他说：“维修权法律已在11个州生效，所以令我惊讶的是，原始设备制造商（OEM）甚至没有提及可维修性。”

这意味着，尽管CES被誉为年度最闪亮新技术的发布盛会，但“除了提高新芯片的速度和性能外，几乎没有展示出真正的创新。更快的芯片对于报时或播放音乐有什么区别吗？在我看来，这非常平庸。”

## 一次性电子产品的问题

在活动中，消费者倡导团体PIRG维修权运动负责人[Nathan Proctor](https://pirg.org/people/nathan-proctor/)谴责了“一次性电池供电产品的绝对罪恶泥潭”，称其是美国废物处理设施火灾的主要原因。

随后，他指出了一款“因其纯粹的荒谬性而引人注目”的电池供电产品——[Lava Brand](https://www.lollipopstar.com/)生产的[一次性电子棒棒糖](https://www.usatoday.com/story/tech/2026/01/09/lollipop-star-speaker-ces-2026/88101256007/)，用Proctor的话说，它“通过下巴的振动播放歌曲”。[Engadget的一段视频](https://www.facebook.com/watch/?v=3345638968926898)指出，每个棒棒糖零售价为8或9美元，而且内含电子元件的棒棒糖棒是不可重复使用的。Proctor利用这款产品传递了一个明确的信息。

Proctor说：“我们需要停止生产如此多的一次性电子产品，它们含有有毒化学物质，需要关键矿物来生产，并且可能烧毁废物处理设施。”

至少有一款产品得到了组织者的认可并引起了我们的注意：联想推出了一款[模块化机箱的笔记本电脑，其可维修性](https://news.lenovo.com/pressroom/press-releases/lenovo-at-ces-2026-smarter-ai-meets-purposeful-business-innovation/)远高于普通笔记本电脑。

《个人电脑杂志》[称赞](https://www.pcmag.com/news/lenovo-new-thinkpad-x1-carbon-gen-14-ces-2026)其为“联想迄今为止最易维修的超便携笔记本电脑”，并指出电池、扬声器、散热风扇和键盘都是模块化的，使其“成为迄今为止最易维修和升级的ThinkPad超便携电脑”。

维修协会执行董事告诉The New Stack，这是一款“我们喜欢并希望支持”的产品。

## “狗屎化”奖及其影响

今年，新增设了一个特别奖项，旨在表彰表现出最大程度“[狗屎化](https://thenewstack.io/cory-doctorow-reveals-how-hed-fix-big-techs-domination/)”的产品，这是一个由作家/EFF活动家[Cory Doctorow](https://www.eff.org/about/staff/cory-doctorow)创造的术语，用来描述科技公司如何无情地滥用其对平台或产品的控制。

Doctorow在颁奖典礼上批评了博世（Bosch）电动自行车上增加的一个复杂的防盗系统，该系统对可维修性产生了严重影响。Doctorow认为，该系统的部件会进行“配对”——首先对照被盗部件数据库进行检查——这可能导致“误报”（例如，在警方拍卖会上购买的部件）。

Doctorow说，问题变成了：你更不信任谁？当然，自行车窃贼可能会偷走你的自行车，但另一个威胁是博世，“如果你以他们不喜欢的方式修理自行车，他们就会报警。”

Doctorow声称，博世还以另一种方式威胁消费者。就像那些如果你使用竞争对手墨盒就会停止工作的打印机一样，“博世随时可以拨动开关，然后除非博世说他们喜欢你使用的部件，否则你将无法使用独立部件进行维修。”

Doctorow分享了一些残酷的现实。“每个开始使用部件配对的制造商都声称他们在打击盗窃和假冒，但最终却将其用于阻止独立维修。”因此，Doctorow用他自己的术语，宣布这款产品为“最差的狗屎化产品”。

Doctorow将[博世部件监控技术](https://www.bosch-presse.de/pressportal/de/en/ces-2026-bosch-ebike-systems-extends-its-digital-theft-protection-for-ebikes-280320.html)的含义推导到一个可能的逻辑极端。自1998年以来，规避版权保护[在美国一直属于重罪](https://www.copyright.gov/legislation/dmca.pdf)，可判处五年徒刑——如果这些法律适用于博世的防盗系统，Doctorow相信，“你修理自行车所坐的牢，会比你偷自行车所坐的牢更久！”

事实上，本周早些时候，维修协会的Gordon-Byrne告诉The New Stack，由于CES缺乏对可维修性和其他以客户为中心的创新的考虑，她曾想建议将其命名为“消费者狗屎化展”。（但她“很难在不带侮辱性的情况下表达出来”。）

## 2026年国际消费电子展最差隐私和安全产品

在活动中，EFF执行董事[Cindy Cohn](https://www.linkedin.com/in/cindy-cohn-9325/)警告了“侵犯隐私的技术”，并指出亚马逊的Ring门铃系统尤其令人发指。Cohn说，它不仅是拍摄你的家门口，现在还增加了面部识别功能，甚至还有“可以部署在停车场和其他地方”的*移动*监控塔。Cohn说，如果这还不够，亚马逊还宣布了一个配套的应用商店，“这将允许人们为门铃开发比亚马逊现有应用更可疑的应用。”

因此，Cohn说，Ring门铃赢得了2026年的奖项，“因为它今年支持的监控能力纯粹是扩张性的”，“因为它加倍侵犯隐私——并支持了‘更多监控总是让我们更安全’的错误观念。”

[![EFF执行董事Cindy Cohn颁发CES 2026最差展品（隐私类）奖——视频截图](https://cdn.thenewstack.io/media/2026/01/fb59276f-eff-executive-director-cindy-cohn-presents-ces-2026-worst-in-show-for-privacy-award-screenshot-from-video.png)](https://cdn.thenewstack.io/media/2026/01/fb59276f-eff-executive-director-cindy-cohn-presents-ces-2026-worst-in-show-for-privacy-award-screenshot-from-video.png)

而在“安全”方面，最差展品连续第二年来自一家中国公司。

Secure Repairs创始人[Paul Roberts](https://www.linkedin.com/in/pfroberts/)指出，Merach的一款内置AI“教练”的跑步机因其令人惊讶的多种安全问题而上榜。它收集财务和个人信息。它收集你的在线行为和网络信息。它收集生物识别数据以及健身和表现数据。但Roberts警告说，根据该公司的隐私政策，它还收集“可以从这些信息中推断出的结论”。

接着，Merach在所有这些数据收集之后，坦率地承认了一个惊人的事实：

“我们无法保证您个人信息的安全。”

## 最难维修和整体最差产品

最差的可维修性产品是三星的一款智能冰箱，iFixit联合创始人[Kyle Wiens](https://www.linkedin.com/in/kylewiens/)指出，三星的冰箱压缩机、制冰机和触摸屏已经存在问题。因此，他对[三星最新款“智能”冰箱](https://www.samsung.com/us/explore/family-hub-refrigerator/overview/)持谨慎态度，因为它没有把手——三星相信其内置的AI功能将随时可用，能够识别语音命令来打开门。

Wiens开玩笑说：“我宁愿把三星冰箱扔出去，也不愿信任它，这就是为什么你不应该把家人的食物托付给它。”

Gordon-Byrne随后在演讲中公布了整体最差展品的评判结果。她说：“我们争论了很久”——但人们普遍不喜欢那个能按指令打开的冰箱。他们认为冰箱门应该保持关闭，以保存食物——而不是在错误的背景噪音下打开，或者在互联网中断或电源问题时出现故障。而且，当靠近冰箱门时，真的有人想看到广告吗？！这甚至还没有考虑最终的担忧：“谁在追踪数据的去向？”

Gordon-Byrne说，它被选为整体最差展品，因为当你权衡所有的缺点和优点时，“从根本上说，你为什么要这个东西？”

## “谁要这个了？”——不必要技术奖

《消费者报告》派出其技术政策总监[Justin Brookman](https://www.consumerreports.org/about-us/our-people/our-experts/justin-brookman/)，将臭名昭著的“谁要这个了？”奖颁给了亚马逊Alexa Plus语音控制的意式咖啡机。Brookman疲惫地指出，“实际上，很多人并不想和他们的咖啡机对话。现在还太早。”

不仅仅是Brookman认为Alexa对于一款已经在[亚马逊上零售价为1445美元](https://amzn.to/4jyHyfj)的咖啡机来说，是一个“一个[有漏洞且不稳定](https://www.theverge.com/tech/853818/bosch-smart-coffee-machine-alexa-plus-upgrade)的助手，需要Prime订阅或月费”。

真正的问题是科技行业偏爱那些“会无故失去关键功能”的昂贵产品。因此，这款咖啡机面临的威胁是，总有一天制造商“不再希望支持咖啡机上的Alexa功能，该功能将被禁用。如果你的咖啡爱好者同伴实际上没有使用个人AI咖啡师功能，那么它可能明天就消失了！”

而在最差展品人民选择奖中，NowThis主编[Michael Vito Valentino](https://www.linkedin.com/in/michaelvitovalentino/)选择了Lepro令人毛骨悚然的[玻璃柜中“AI伴侣”](https://www.lepro.com/pages/lepro-ami-coming-soon?srsltid=AfmBOoo6uk74IUDRKkObvsgO99PNJGPl1eRXjKc4S2PSOg_KS491SFiB)，名为AMI。

也许所有这一切都证明，尽管国际消费电子展雄心勃勃，但真正为消费者最佳利益着想的是维修倡导者。他们生动的节目由YouTube科技实验者[Simone Giertz](https://www.youtube.com/c/simonegiertz)主持，她可能为那些表彰安全、隐私、环境影响和可维修性方面最差产品的奖项提供了最佳描述。

最终，它们是“没人会想要的”奖杯。
