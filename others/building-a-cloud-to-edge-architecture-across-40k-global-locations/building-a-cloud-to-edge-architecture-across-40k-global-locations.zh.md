在任何分布式规模的边缘都需要新的管理方法，以应对[边缘的特殊性](https://thenewstack.io/edge-computing/)，例如连接性差、硬件受限或现场技术支持缺乏。

但这并不能免除企业 IT 团队的责任，他们承受着巨大的压力，需要在公司无论多么庞大、分散或存在问题的业务覆盖范围内，维护可靠、一致的数字服务。毕竟，数字体验与客户满意度、卓越运营和业务目标息息相关，[维护这些服务](https://thenewstack.io/ebooks/kubernetes/running-virtual-machines-on-kubernetes-a-practical-guide-for-enterprise-migrations/)是许多企业 IT 团队的首要职责。

在数千（甚至数万）个地点实现这一服务水平是一个巨大的挑战，正如全球连锁餐厅 Taco Bell、KFC、Pizza Hut 和 Habitat Burger 的母公司 YUM! Brands 所发现的那样。它发现的解决方案是一种模块化、基于容器且原生于边缘的架构，该架构跨其 40,000 多家全球门店标准化了数字运营。
