[Rust Foundation](https://rustfoundation.org/) 周四宣布推出 [Rust Foundation Trusted Training (RFTT)](https://rustfoundation.org/rftt/) 项目，这是一项由负责该语言管理的非营利组织支持的，针对 [Rust](https://thenewstack.io/rust/) 培训提供商的全新正式认证。

创始成员由五家机构组成：Mainmatter、Integer 32、Wyliodrin、Doulos 和 [Ferrous Systems](https://thenewstack.io/rust-gets-its-missing-piece-official-spec-finally-arrives/)——它们都是 Rust 培训市场的资深参与者。

Rust Foundation 执行董事兼首席执行官 Dr. Rebecca Rumbul 告诉 *The New Stack*，Rust Foundation 相信该项目将建立信任，确保那些自称是专家的人确实名副其实。

“随着 Rust 在系统编程、嵌入式开发到安全关键基础设施等各行业的应用加速，Rust 培训市场也随之迅速增长，” Rumbul 说。“Rust Foundation Trusted Training 项目为学习者、雇主和更广泛的开发者社区提供了一个评估 Rust 培训质量的可信信号。”

> “Rust Foundation Trusted Training 项目为学习者、雇主和更广泛的开发者社区提供了一个评估 Rust 培训质量的可信信号。”

Rust 的一大痛点是人们普遍认为其[学习难度大](https://thenewstack.io/google-busts-confirms-common-myths-about-rust-programming/)。

2016 年专门围绕 Rust 成立的公司 Integer 32 的联合创始人 Carol Nichols 在一份声明中表示：“我们很高兴能帮助更多人快速掌握 Rust。”

## 认证流程如何运作

RFTT 认证从五个方面进行评估：信誉与道德规范、内容质量、讲师能力、透明度与可访问性，以及评估与反馈。Rumbul 表示，每份申请都由一个活跃的 Rust 培训专业人士组成的委员会根据公开的评分标准进行独立评分。最终决定由 Rust Foundation 高层领导及其董事会做出。

认证有效期为两年，每年费用为 3,000 美元，另需一次性缴纳 500 美元的申请费以涵盖初步审查费用。续期不再收取申请费。微型企业可享受优惠费率。费用标准已在 rustfoundation.org/rftt 公布。

Rumbul 明确表示，该项目旨在为基金会创造收入，但强调其运营而非营利为导向。她说，收费标准是为了覆盖开展可信审查项目的管理成本，而非创造盈余，并补充说，认证是根据公开标准评估的，而不是通过付费获得的。

## 资格与范围

要获得资格，机构必须是 Rust Foundation 的成员，且业务运营至少两年，并至少提供了一年的 Rust 培训。独立培训师和微型企业（定义为员工人数不超过 3 人且年收入低于 10 万美元的组织）可以联系 training@rustfoundation.org 逐案探讨资格。

该项目对非英语培训提供商开放。其他语言的材料在提交审查时需附带英文翻译，并且透明度和可访问性标准中，地理和语言的多样性会被计为加分项。

自定进度课程和在线课程均在范围内。基金会表示，对于[人工智能生成的学习工具](https://www.harvard.edu/ai/teaching-resources/)，目前没有更广泛的立场。

## 利益冲突问题

然而，该项目的治理结构引发了一些疑问。在帮助制定认证标准的四名指导委员会成员中，有三名——Mainmatter 的 Luca Palmieri、Integer 32 的 Carol Nichols 和 Wyliodrin 的 Alexandru Radovici——是创始群体中机构的创始人。

> “我们不会假装专业知识与申请者池之间没有重叠。确实有重叠，所以我们构建了流程来控制它。”

Rumbul 承认了这种重叠。

“我们不会假装专业知识与申请者池之间没有重叠，” 她告诉 *The New Stack*。“确实有重叠，所以我们构建了流程来控制它。”

保障措施包括标准是公开且基于能力的。

此外，Rumbul 说：“没有成员审查自己的申请，评审员也不决定谁能通过。” 相反，他们根据应用于每位申请者的统一、公开的必要条件进行评估，并仅提出建议；评审员提出的疑虑会触发委员会的进一步审查，而不是自动拒绝，最终决定权在于 Rust Foundation 的高层领导。因此，审查其他提供商意味着应用统一的公开标准向我们提出建议，而不是对竞争对手行使自由裁量权。

## 并非所有人都能通过

基金会证实，并非所有申请者都在第一次尝试时就达到标准。一些申请者获得了具体的反馈和重新获得资格的途径。基金会出于公平和保密考虑，不会披露个别申请者的结果或结果统计。

> “指名道姓是不公平的，也会打消机构申请的积极性。我们的目标是帮助提供商达到标准。”

“指名道姓是不公平的，也会打消机构申请的积极性，” Rumbul 说。“我们的目标是帮助提供商达到标准。”

## 下一步计划

Rumbul 将第一年的成功主要描述为信任层面的成功，而非增长层面的成功。她说，该项目是围绕质量信号构建的，而非收入，而且 Rust 培训提供商的数量并非无穷无尽。

个人或课程级别的认证在长期路线图中，但取决于容量。目前，该称号适用于提供商的整体培训产品，而不是个别课程或学习者。

RFTT 提供商列表和完整的项目标准可在 rustfoundation.org/rftt 获取。基金会也正在以滚动方式接受新申请。
