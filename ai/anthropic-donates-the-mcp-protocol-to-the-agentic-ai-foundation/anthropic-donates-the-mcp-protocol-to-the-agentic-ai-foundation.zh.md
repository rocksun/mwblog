Anthropic 的模型上下文协议 (Model Context Protocol, MCP) [仅一年时间](https://blog.modelcontextprotocol.io/posts/2025-11-25-first-mcp-anniversary/)，但已迅速成为大型语言模型 (LLM) 在智能体循环中访问数据、工具和服务的默认协议。现在，Anthropic 正将其移交给在 Linux 基金会旗下新成立的开源基金会：智能体AI基金会 (Agentic AI Foundation, AAIF)。

该基金会启动时，还带来了另外两个项目：来自 Block 的 AI 智能体框架 [goose](https://block.github.io/goose/)，以及来自 OpenAI 的 [AGENTS.md](https://agents.md/)，后者已成为向编码智能体提供编码指南、构建步骤等上下文的事实标准格式。

将 MCP 移交给一个中立基金会酝酿已久（并且比原计划提前了一天）。Linux 基金会的多位代表已暗示此举一段时间，正如 OpenAI 的 Nick Cooper 告诉我的那样，基金会实际的启动过程大约在四个月前正式开始。“但在我看来，移交给 Linux 基金会是必然会发生的事情。我们只是不知道具体什么时候，”他说道。

鉴于 MCP 和 AGENTS.md 的普及，特别是，新基金会的成员囊括了前沿模型公司、智能体AI编码领域的领导者以及许多使用这些工具的初创公司，可谓精英荟萃。

新基金会的白金会员包括 [亚马逊](https://aws.amazon.com/?utm_content=inline+mention)、Anthropic、Block、Bloomberg、Cloudflare、Google、Microsoft 和 OpenAI。其他黄金级别会员包括 Adyen、Cisco、Datadog、Docker、[IBM](https://ad.doubleclick.net/ddm/trackclk/N1114924.4097665THENEWSTACK/B33037161.434371219;dc_trk_aid=627496700;dc_trk_cid=227599223;dc_lat=;dc_rdid=;tag_for_child_directed_treatment=;tfua=;gdpr=$%7BGDPR%7D;gdpr_consent=$%7BGDPR_CONSENT_755%7D;ltd=;dc_tdv=1?utm_content=inline+mention)、JetBrains、Okta、[Oracle](https://www.oracle.com/developer?utm_content=inline+mention)、Runlayer、SAP、[Snowflake](https://www.snowflake.com/?utm_content=inline+mention)、[Temporal](https://temporal.io/?utm_content=inline+mention)、Tetrate 和 Twilio 等。白银会员包括 [Chronosphere](https://chronosphere.io/?utm_content=inline+mention)、Cosmonic、Elasticsearch、Hugging Face、Kubermatic、Pydantic、[Spectro Cloud](https://www.spectrocloud.com/?utm_content=inline+mention)、SUSE、Uber、WorkOS 和 ZED。

另一家白银会员 Obot.ai 已将其 MCP 开发者峰会活动捐赠给基金会，这意味着 AAIF 的首个官方活动将是明年四月在纽约举行的 [MCP开发者峰会](https://events.linuxfoundation.org/mcp-dev-summit-north-america/)。

“MCP 最初是一个内部项目，旨在解决我们团队面临的问题。当我们在 2024 年 11 月将其开源时，我们希望其他开发者也能发现它像我们一样有用，”Anthropic 的首席产品官 Mike Krieger 说道。“[...] 将 MCP 作为 AAIF 的一部分捐赠给 [Linux 基金会](https://training.linuxfoundation.org/training/course-catalog/?utm_content=inline+mention)，确保它作为 AI 的关键基础设施保持开放、中立和社区驱动。我们将继续致力于支持和推进 MCP，凭借 Linux 基金会管理驱动互联网项目数十年的经验，这仅仅是个开始。”

尽管基金会以这批初始贡献项目启动，但其明确的目标是随着时间推移，在智能体AI领域增加更多的开源工具。例如，如果很快有新的智能体框架加入，我也不会感到惊讶。

与所有类似的 Linux 基金会一样，这里的核心思想是为这些基础项目提供一个中立的归宿。“将 MCP 放置在一个供应商中立的基金会中，确保开发者可以放心地投资于这个通用标准，知道它将保持开放、可互操作且由社区驱动。我们期待与 Anthropic 一起继续为 MCP 项目做出贡献，以解锁智能体AI应用和我们客户的新可能性，”亚马逊云计算服务智能体AI副总裁 Swami Sivasubramanian 解释道。

OpenAI 的 Cooper 告诉我，该公司捐赠 Agents.MD 的动机是相似的。“这与我们所有的开源技术工作非常相似，无论是 Agents.MD、其他开源举措，还是我们围绕 MCP 的参与，”他说。“我们只是希望推动创建一个开放、中立的空间，在那里我们可以从其他人那里获得有意义的贡献，获得开发者社区和用户社区的反馈，并真正帮助构建和完善所有这些协议，使其真正具有重要意义。”
