OpenAI推出每月100美元的ChatGPT Pro层级，填补了20美元与200美元间的空白，旨在吸引高频使用Codex的开发者，并直接竞争Anthropic。

> 译自：[OpenAI's new $100 tier targets developers hitting Codex (and Claude Code) limits](https://thenewstack.io/openais-new-100-tier-targets-developers-hitting-codex-and-claude-code-limits/)
> 
> 作者：Frederic Lardinois

周四，OpenAI [宣布](https://x.com/OpenAI/status/2042295688323875316?s=20)了新的 [每月 100 美元的 ChatGPT Pro 档位](https://chatgpt.com/pricing)，虽然该档位向所有人开放，但专门针对 Codex 用户。

OpenAI 的 Codex 应用和用于智能代理编程（agentic coding）的 CLI 工具在公司内部大获成功，拥有超过 300 万活跃用户，过去三个月增长了 5 倍，月环比增长 70%。但直到现在，个人开发者在每月 20 美元的 ChatGPT Plus 计划和昂贵得多的每月 200 美元 Pro 计划之间面临着巨大的工作量差距。

这个新档位将为开发者提供比每月 20 美元的 Plus 档位多 5 倍的 Codex 使用量（而 200 美元的计划提供 20 倍），并能访问之前仅向最昂贵计划订阅者提供的 Pro 模型和深度研究能力。此计划的用户还将获得实验性功能的早期访问权限。

![](https://cdn.thenewstack.io/media/2026/04/b874ea71-screenshot-2026-04-09-at-11.23.54-am-1024x617.png)

这听起来可能有点耳熟，因为 Anthropic 也提供几乎相同的计划，其每月 100 美元的 Max 档位同样提供 5 倍的使用量，每月 200 美元的计划提供 20 倍。

Anthropic 将其部分企业功能（如 PowerPoint 中的 Claude）锁定在 Max 计划之后，尽管所有付费用户都可以访问其旗舰模型（[至少目前如此](https://thenewstack.io/anthropic-claude-mythos-cybersecurity/)）。对于 Anthropic 来说，额外的使用量也很可能流向其编程工具，尤其是现在它已经关闭了 OpenClaw 和类似工具的访问。

开发者们最近一直在抱怨他们[达到 Claude Code 使用限制](https://thenewstack.io/claude-code-usage-limits/)的速度有多快。这在一定程度上是 Anthropic 的失误，但最近的普遍基调是，Codex 即便是在每月 20 美元的计划中，提供的限制也更为慷慨。显然，OpenAI 已经意识到了这一点，现在旨在通过这个新的每月 100 美元的计划获利，因为它希望保持其通过 Codex 建立的势头。

在宣布这一新功能的致媒体邮件中，该公司对此也表现得相当直白：“与 Claude Code 相比，Codex 在各付费档位中提供了更高的单位美元编程容量，这种差异在活跃编程使用期间表现得最为明显。”

OpenAI 指出，每月 20 美元的 ChatGPT Plus 仍然是日常使用的最佳选择。

按照惯例，OpenAI 将在未来几周内临时提高这一新计划的使用限制，并在 5 月 31 日之前为每月 100 美元的 Pro 用户提供 10 倍的使用量。
