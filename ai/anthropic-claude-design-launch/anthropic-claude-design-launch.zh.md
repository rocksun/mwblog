Anthropic Labs，作为 Claude 开发者的 AI 安全与研究[部门](https://www.anthropic.com/news/introducing-anthropic-labs)，于周五推出了 [Claude Design](https://claude.ai/design)。这是一项处于研究预览阶段的新服务，旨在生成完善的设计系统、网站原型、交互式网站、幻灯片、单页宣传页以及类似的交付物。在以往，你通常需要聘请设计师或使用 Canva 或 Figma 等产品才能开始这些工作。

这次发布并不令人意外。本周早些时候，《The Information》就曾报道 Anthropic 正在开发一款设计工具。与此同时，Anthropic 的首席产品官 Mike Krieger 于本周早些时候辞去了 Figma 董事会的职务。Figma 的股价在过去 12 个月内已经下跌了近 50%，在 Claude Design 发布后立即又下跌了 5%。

## 让设计师在 Token 耗尽前尽情探索

Anthropic 指出，Claude Design 旨在为“设计师提供广泛探索的空间，并为其他人提供一种制作视觉作品的方式”。当然，前提是他们的 Token 还没用完。如果你拥有 Claude 的付费计划（Pro、Max、Team 和 Enterprise），Claude Design 会有专门的每周额度，而且该系统对 Token 的消耗非常大。在构建了一个设计系统和基于该系统的业务网站原型，并进行了几次调整和一个说明视频后，我已经使用了超过 50% 的每周配额。一旦达到上限，你就需要支付即用即付的 Token 费用。

![](https://cdn.thenewstack.io/media/2026/04/d84c330b-screenshot-2026-04-17-at-9.41.24-am-1024x297.png)

公平地说，你可以选择只构建线框图而不是精美的视觉原型，线框图（或基础幻灯片）可能会消耗更少的 Token。但使用 Claude 这种工具的一半乐趣就在于能够以以前不可能的方式进行实验。

## 使用 Claude Design

要开始使用，该服务会为你构建一个设计系统。你可以向它提供你的代码库和现有的设计文件（或实际上任何其他视觉资产），它就会开始工作。这个过程的一个优点是，它几乎在每个方面都让你拥有最终决定权。它会选择颜色和字体，并为你创建设计元素，但你可以批准这些元素或要求修改。

![](https://cdn.thenewstack.io/media/2026/04/0d66c008-screenshot-2026-04-17-at-8.52.44-am-1024x793.png)

*Claude Design 以 Joshua Topolsky 风格重新设计的 New Stack 首页（来源：The New Stack）。*

你也可以在看到最终结果后返回进行修改。

为了让 Claude 微调结果，你还可以对预览中的特定设计元素进行评论，这与 OpenAI 为构建视觉资产而推出的[新版 Codex](https://thenewstack.io/openais-superapp-takes-shape/) 类似。但 Claude Design 更进一步，它允许你在设计上进行绘画并直接编辑某些元素（如背景颜色和字体）。但这里最有趣的选项是让模型生成你希望看到的滑块和选项，以便实时调整设计，而无需向 Claude 请求更改。

![](https://cdn.thenewstack.io/media/2026/04/6f80d7ba-screenshot-2026-04-17-at-9.23.36-am-1024x388.png)

设计师将如何使用它还有待观察，但 Anthropic 认为它非常适合创建逼真的原型、产品模型、路演幻灯片和营销素材，同时也适用于前沿设计——设计师可能希望构建由代码驱动的原型和常规的设计探索，而这些在以前可能过于耗时或繁琐。

## 导出

一旦在 Claude Design 中完成工作，它可以将设计导出为 PDF、PowerPoint 幻灯片或独立的 HTML 文件。此外，还可以选择将它们发送到 Canva（不是 Figma），或者将设计移交给 Claude Code，将其转化为实际运行的产品。

如果该服务还能提供直接与团队共享运行原型的选项而无需导出，那就更好了，但毕竟这只是一个预览版。Anthropic 团队在公告中还提到：“在接下来的几周内，我们将简化与 Claude Design 构建集成的过程，以便你可以将其连接到团队已经在使用的更多工具中。”
