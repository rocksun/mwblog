OpenAI 更新了 Codex，为其引入了操作 macOS 应用、网页浏览及长期记忆等 Agent 能力，旨在直接竞争 Anthropic 的 Claude Code，提升自动化开发效率。

> 译自：[OpenAI's Big Codex Update Is a Direct Shot At Claude Code - Slashdot](https://developers.slashdot.org/story/26/04/16/2040206/openais-big-codex-update-is-a-direct-shot-at-claude-code)
> 
> 作者：[no-author]

OpenAI 正在[为 Codex 更新更具智能体（agent）属性的功能](https://www.theverge.com/ai-artificial-intelligence/913034/openai-codex-updates-use-macos)，将其定位为 Anthropic 旗下 Claude Code 更直接的竞争对手。一些新功能包括操作 macOS 桌面应用、在应用内浏览网页、生成图像、使用新的办公插件，以及记住过去任务中的有用上下文。据 The Verge 报道：

*OpenAI 在一篇宣布更新的[博客文章](https://openai.com/index/codex-for-almost-everything/)中表示，Codex 现在将能够操作你电脑上的桌面应用。它可以在后台运行，这意味着它不会干扰你在其他应用中的工作，并且多个智能体可以并行工作。对于开发者，OpenAI 表示“这对于测试和迭代前端更改、测试应用或在不提供 API 的应用中工作非常有帮助”。该功能将从今天开始向登录 ChatGPT 的 Codex 桌面应用用户推出，最初仅限于 macOS。OpenAI 没有说明何时会扩展到其他操作系统。欧盟用户也需要等待，该公司补充说更新将“很快”向那里的用户推出。*

*Codex 还获得了通过 gpt-image-1.5 生成和迭代图像的能力，以及针对 GitLab、Atlassian Rovo 和 Microsoft Suite 等工具的新插件，并支持通过应用内浏览器进行原生网页浏览，“你可以在页面上直接评论，向智能体提供精确指令”。OpenAI 还表示，自动化任务将变得更加容易，用户能够重复使用现有的对话线程，且 Codex 现在可以为自己安排未来的工作，并自动唤醒以继续执行长期任务。Codex 还将获得记忆功能，使其能够记住过去经验中的有用上下文，例如个人偏好、修正以及需要时间收集的信息。OpenAI 表示，希望这个作为预览版发布的选入功能（opt-in feature）能帮助未来的任务更快完成，并达到以前需要详细自定义指令才能实现的质量。个性化功能将“很快”向企业版、教育版和欧盟用户推出。*
