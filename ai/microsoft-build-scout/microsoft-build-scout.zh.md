在周二于旧金山举行的微软 Build 大会上，该公司首次推出了 Microsoft Scout。这是一款全新的个人工作智能体，它能利用你现有的工具来理解你的工作方式，并主动处理日常任务。

## Scout 的功能

Microsoft Scout 目前已面向所有 Frontier 客户开放，更多细节和更广泛的推广将在稍后公布。它构建在 OpenClaw 和 WorkIQ（Microsoft 365 Copilot 的智能层）之上。

微软表示，该个人智能体将与用户现有的工具集（如 Teams、Outlook 等）协同工作。

> Scout “理解你的工作方式”，并且无需吩咐即可主动处理日程冲突、准备会议以及其他常规任务。

这家科技公司表示，Scout “理解你的工作方式”，并且无需吩咐即可主动处理日常工作，例如解决日程冲突、准备会议以及其他常规任务。

## 为什么是现在？

微软在发布的新闻稿中表示，它相信“你的智能体应该反映你的思考和运作方式”，并考虑到你更广泛的工作环境（包括工作流、业务逻辑和机构知识），并指出这种上下文应该延伸到全天候运行的自主智能体。

> Microsoft Scout 目前仅向 Frontier 客户开放。微软表示，作为更广泛推广计划的一部分，它将很快分享更多信息。

Scout 的首次亮相伴随着该公司在为期两天的 Build 大会开幕时发布的一系列其他声明，其中包括：Microsoft IQ，这是一个新的上下文层，可让智能体立足于企业和世界知识，今天已在 GitHub Copilot、Microsoft Foundry 和 Copilot Studio 中提供；WebIQ，一个 AI 优先的网络搜索技术栈，微软称之为“你能提供给智能体的最快真实世界落地（grounding）方式”；此外，还有来自 Microsoft AI Superintelligence 团队的七款全新自研模型，包括该公司的首个推理模型 MAI-Thinking-1。

在 Build 大会上，微软强调了新智能体时代（agentic era）正在发生的变化，指出开发人员现在需要“无处不在的智能”以及一个支持他们从云端到边缘的完整技术栈。如果 Scout 能够理解团队的工作方式，并为至少一些日常任务提供主动支持，那么它在帮助团队更快构建智能体系统方面将起到辅助作用。

在 Scout 发布之前，最近泄露了[谷歌的 Remy](https://thenewstack.io/google-remy-agent-infrastructure/?utm_source=chatgpt.com)，传闻这家科技巨头正在开发这款 OpenClaw 风格的智能体，可以代表用户执行操作。Remy 被怀疑是基于 Gemini 的个人智能体。目前判断 Microsoft Scout 是否以及如何与传闻中的 Remy 进行比较还为时过早，但现有的有限信息表明，两家公司都在朝着超越“回答问题或总结内容”的个人智能体方向迈进。
