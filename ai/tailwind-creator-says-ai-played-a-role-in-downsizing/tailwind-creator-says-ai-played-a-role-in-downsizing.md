Tailwind CSS裁员75%工程师，创始人Wathan归咎AI致收入降。后获Vercel等赞助，Wathan感谢支持，称财务已稳。AI影响文档流量，间接损及付费产品销售。

> 译自：[Tailwind Creator Says AI Played a Role in Downsizing](https://thenewstack.io/tailwind-creator-says-ai-played-a-role-in-downsizing/)
> 
> 作者：Loraine Lawson

热门的 CSS 框架 Tailwind CSS 于周一解雇了三名工程师——占其工程团队的 75%。在周三发布到 X 平台的一段[个人播客](https://x.com/adamwathan/status/2008909129591443925)中，Tailwind 的创建者 [Adam Wathan](https://github.com/adamwathan) 声称，人工智能在该开源项目收入下降中扮演了重要角色。

Wathan 谈到裁员时说：“我认为分享这个百分比很重要，因为如果只说‘哦，我们不得不解雇三个人’，听起来好像没做什么大调整，但实际上，我们有四名在职工程师，现在只剩一名了。所以这是一个巨大的变化。”

但到周四下午，情况发生了转变，前端云公司 [Vercel](https://thenewstack.io/vercel-marketplace-offers-agentic-ai-building-blocks/)、Postgres 开发平台 [SupaBase](https://thenewstack.io/how-supabase-is-building-its-platform-engineering-strategy/) 和 Google AI Studio 都在推文中表示将赞助该项目。Tailwind CSS 自 2025 年夏季以来一直设有[赞助计划](https://tailwindcss.com/sponsor)。

Wathan 周五通过 [Twitter 表示，过去 24 小时内的支持让他“不知所措”](https://x.com/adamwathan/status/2009340684210159812)，这在他录制关于项目财务状况的播客时是始料未及的。

他写道：“非常感谢所有给予鼓励、新加入的赞助商以及利用其平台提供帮助的人们，我深受感动。”

尽管如此，他补充道：

“我们现在很安心，合作伙伴计划等措施正在很大程度上帮助我们更直接地为框架提供资金，而不是销售更容易面临颠覆风险的产品。我们不需要任何人来拯救，我们有充足的时间和空间去尝试新想法并看看什么会奏效 🙏🏻 我乐观地认为我们能找到正确的方法来推动事情发展，但即使我们只是一个预算比过去更少的小型企业，那也完全没问题 ❤️。”

Wathan 周三表示，他知道——并公开承认——收入正在下降，但他曾认为收入已经稳定。随后，他在假期期间查看了预测数据。

他说，当他意识到问题范围时，“感到措手不及”。

他说：“如果什么都不改变，那么在大约六个月内，我们将无法履行工资支付义务，这是一个非常糟糕的现实。”“在假期期间，我们不得不做出缩减团队规模的实际决定，这是公司目前能负担得起的，因为如果现在不这样做，那么我们就无法为员工提供丰厚的遣散费，让他们有时间寻找新职位。”

这对 Tailwind CSS 来说是一个令人惊讶的发展，Wathan 在 2020 年曾将其誉为[一个开源成功案例](https://adamwathan.me/tailwindcss-from-side-project-byproduct-to-multi-mullion-dollar-business/)。

他当时写道：“我们 Tailwind UI 的收入也即将突破 200 万美元，这是我们的第一款商业 Tailwind CSS 产品，大约五个月前发布——距离第一版 Tailwind CSS 发布还不到两年。”

如今，Tailwind CSS 是[前端开发](https://thenewstack.io/introduction-to-frontend-development)领域迄今为止最受欢迎的 CSS 框架。在 [Devographic](https://www.devographics.com/) 的 [2025 年 CSS 现状调查](https://2025.stateofcss.com/en-US/)中，5,506 名受访者中有 [51%](https://2025.stateofcss.com/en-US/other-tools/) 称其为首选框架。Bootstrap 以 30% 的回应位居第二。

发生了什么变化？Wathan 认为，部分原因是人们使用人工智能来总结 Tailwind 的文档，而不是访问实际文档——那里是他们了解 Tailwind 付费产品的地方。

他说：“整个 AI 事件显然是一把双刃剑，因为 AI 是我们业务陷入困境的一个巨大原因，尽管它让 Tailwind 比以往任何时候都更受欢迎。”

他在[一个拉取请求回复](https://github.com/tailwindlabs/tailwindcss.com/pull/2388)中表示，自 2023 年以来，Tailwind 文档的流量下降了 40%。

> “整个 AI 事件显然是一把双刃剑，因为 AI 是我们业务陷入困境的一个巨大原因，尽管它让 Tailwind 比以往任何时候都更受欢迎。”
> **— Tailwind 创建者 Adam Wathan**

Wathan 在他的播客中说：“那是我们的全部分发渠道，而为框架——这个免费的开源框架——工作的任何人提供报酬的唯一途径，就是人们购买产品。”“如果更多的人因为依赖 [LLM](https://thenewstack.io/from-agi-hype-to-engineering-reality-the-future-of-llms/)（大型语言模型）抓取 markdown 端点而停止查看文档……那只会让更少的人了解到付费产品，因为 markdown 文件，就其目前的形式而言，并没有向人们告知商业产品的存在。”

Wathan 在拉取请求中表示，他希望找到一种方法来提供 LLM 优化的文档，以避免情况恶化，但他目前无法优先处理此事。他警告说，在人员如此有限的情况下，关闭拉取请求和开发新功能可能会更加困难。

Wathan 在播客中透露，Tailwind 剩下的团队包括 Wathan 和另外两名所有者，一名工程师以及一名兼职运营人员。最后，他强调了刚刚解雇的三名工程师的资历。
