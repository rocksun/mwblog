[Base44](https://base44.com/) 是一家由 AI 驱动的应用构建平台，近期发布了其首个专有 AI 模型 Base One。这是一个经过微调的开源 [LLM](https://thenewstack.io/llm/)，专门针对 [氛围编程](https://thenewstack.io/vibe-coding-ceos-executives-ai-agents/) Web 应用进行了训练。

该公司声称这是行业首创：即首个推出自有模型的应用创建平台。不过，这一说法带有一定的局限性。

Base44 创始人兼 CEO [Maor Shlomo](https://www.linkedin.com/in/maor-shlomo-1088b4144/) 在接受 *The New Stack* 采访时确认，Base One 是基于现有开源基础进行的微调，并非从零开始训练的模型。（“专有模型”一词在整个行业中被广泛使用。）

## 训练方法

其训练方法由 [强化学习](https://thenewstack.io/reinforcement-learning-pioneers-honored-with-acm-turing-prize/) 驱动。他表示，Base44 让模型在真实的平台任务中反复运行，构建并编辑应用程序，将输出结果评定为“好”或“坏”，并将该信号反馈给模型以更新其权重。Shlomo 指出，训练数据是混合来源的，大部分是通过在 Base44 平台上运行模型并使用强化学习反馈来改进，同时也结合了一些合成数据。他补充道，该模型目前已投入生产并为用户提供服务。

Shlomo 表示，像 Claude 或 GPT 这样的通用前沿模型被训练成“全能选手”，从 Web 开发到 GPU 内核编写，再到 C++ 和中文诗歌，无所不包。Base44 的赌注在于，一个专门用于将自然语言提示转换为可运行 Web 应用，且内置了设计和产品决策的模型，将在其特定使用场景中优于通用模型。

“对于许多用例来说，它（前沿模型）可能不是最佳选择，”Shlomo 在谈到前沿模型时说，“尤其是当你能够控制、调整并使模型契合你的需求时。”

拥有模型堆栈还可以节省开支。Base44 可以直接控制计算和推理成本，Shlomo 表示，这是 AI 原生业务中最大的成本驱动因素之一。因此，这减少了其产品核心部分对外部供应商的依赖。

## 加速开发

事情的发展比 Shlomo 预期的要快。他说，一年前他无法想象 Base44 会这么快就开始训练自己的模型。两个因素推动了这一进程：开源模型质量的快速成熟，为微调提供了坚实基础；以及 Base44 自身的发展轨迹，产生了微调有效运作所需的数据量和流量。

Wix 去年以 8000 万美元收购了 Base44。Shlomo 表示，Wix 收购后不到一年，公司收入从 300 万美元增长到 1.5 亿美元，这一数字已由 Wix 公开。

该模型的开发是 Base44 与 Wix 现有机器学习和数据科学团队之间的合作，后者曾构建过 Wix Harmony。模型本身针对 Base44 的代理框架进行了调整，其中包括其工具、指令以及代理在平台内的操作。

## 竞争

氛围编程领域的竞争对手包括 Lovable、Replit、Bold 和已进入该类别的 Figma。Shlomo 表示，他并不指望新模型的推出会从根本上改变竞争格局，但随着 Base44 对未来版本的迭代，他将其视为质量和效率上的优势。

Base One 只是 Shlomo 所称的一系列发布中的第一个，未来计划发布更大的模型并进行更深度的产品集成。

“我们在训练模型这条赛道上才刚刚起步，”他说。
