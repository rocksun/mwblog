OpenAI与Work Louder合作推出售价230美元的Codex Micro机械小键盘。该产品配备专门的代理控制键和指示灯，旨在优化AI编码工具的使用体验，并支持自定义功能以提升工作效率。

> 译自：[OpenAI's first gadget is the $230 Codex Micro macropad](https://thenewstack.io/openai-codex-micro-macropad/)
> 
> 作者：Frederic Lardinois

OpenAI 的 Codex 即将迎来 [900 万用户](https://x.com/thsottiaux/status/2077271889626706300?s=20)，其中至少有一部分用户很快将获得一种使用 OpenAI 代理式编码工具的新方式：一款由 OpenAI 与 [Work Louder](https://worklouder.cc) 合作开发的编程机械宏键盘。

Codex Micro 于周三限量发布，售价 230 美元，它是 Work Louder 低剖面 [Creator Micro 2](https://worklouder.cc/creator-micro-2) 键盘的变体（该公司此前曾与 [Figma 合作](https://medium.com/design-bootcamp/figma-x-worklouder-figma-creator-micro-keyboard-setup-and-review-fb2f18391599)推出了类似的宏键盘，目前也为 Framer 用户提供了特别版）。

现已开放预订，售完即止。

## OpenAI 的首款硬件产品

根据你对硬件的定义，这确实是 OpenAI 的首款硬件产品，但值得注意的是，这并非传说中与 Jony Ive 的合作项目。那是[完全不同的产品](https://techcrunch.com/2026/07/14/openais-first-hardware-device-is-reportedly-a-screenless-speaker-that-can-move/)。这本质上只是 Work Louder 之前键盘的一个换壳版本。

至于硬件本身，Codex Micro 看起来使用了 Work Louder 自有的低剖面机械轴体，考虑到 Work Louder 专注于低剖面键盘，这并不令人惊讶。

买家可以在静音轴和段落轴之间进行选择，并可以通过 USB-C 线缆或蓝牙连接 Codex Micro。

## 为你的代理准备的按键

与 Creator Micro 2 相比，OpenAI 版本更贵，但这是有原因的。

除了旋钮和摇杆外，它还具有 OpenAI 和 Work Louder 所称的“代理键”。这是键盘顶部的六个按键，配有透光键帽，可以根据你代理的状态以不同颜色亮起。单按选择与按键关联的代理，双按将其调至前台，颜色让你一目了然地了解正在发生的事情：白灯表示代理空闲，蓝灯表示正在思考，绿灯表示已完成，红灯表示出现错误。

由于 Codex Micro 上的所有内容都可以使用 Work Louder 的专有 Input 工具进行自定义，用户可以选择这六个代理键代表 Codex 中的固定任务、最近任务或需要人工输入的优先级任务。为了实现完全控制，还可以选择将特定代理分配给某个按键。

![](https://cdn.thenewstack.io/media/2026/07/a6f7e1e3-codex-micro-keys--1024x373.gif)

图片来源：OpenAI。

在宏键盘的底部，有六个常规按键，默认分配给用户接受或拒绝更改以及分支线程。最底行较大的按钮被设置为激活一键通话功能——因为在硅谷，通过语音使用 Codex 似乎是启蒙的第三步。

Codex Micro 具有一个 LED 灯光的亚克力边缘，因此当 Codex 录音时，它会亮起，让你知道麦克风已开启。

至于摇杆，默认情况下，它控制 Codex 中的编辑器栏，而旋钮让你进行选择。但和 Codex Micro 的所有功能一样，也很容易将这些映射到其他功能上。

所有这些按键都可以重新映射，键帽也可以更换。Codex Micro 附带 32 个额外的键帽，包括用于你所有氛围编程乐趣的“yolo”和“yeet”键帽。

## 超越 Codex

虽然这里的重点是 Codex，但 Work Louder 的 Input 应用程序允许你创建多达五个额外的层，并将它们用于任何其他应用程序。

你需要 Codex Micro 吗？不需要。

它是一个有趣的小玩具吗？当然。

它会让你效率更高吗？不清楚。

它值 230 美元吗？只有你自己能决定。
